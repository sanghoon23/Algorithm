/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
// * 문제
GridSearch

- Input
n x k , 행 과 열의 수
G = 해당 행렬

s x t , 행과 열의 수 
P = 찾을 해당 행렬

- Output
"Yes" or "No" 출력

G 의 행렬에서 P 에 해당하는 행렬이 존재하는지 판단하라는 문제.

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
// * 해결 방법

1. 먼저 G(전체) 를 돈다.

2. 돌면서 P 에 해당하는 문자열 첫번째가 같다면,
FoundStr 함수를 실행한다.

3. bool Found(string G, int r, int c, string P ) 로
for(P, i, j ) P 를 돌면서
찾은 행 r, 열 c 을 이용해서 같은지 판별한다.
같지 않다면 false 를 return 한다.


/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
// * 시행착오

답은 쉽게 나왔는데, 속도가 느렸다.
그래서 처음 찾은 G(전체) 행렬에서 P.size() 를 빼버렸다.
왜냐하면 P.size 보다 작은 곳은 찾을 의미가 없다.

또 함수를 실행하면서 string 매개변수 G, P 가 복사가 일어났다.
그래서 Pointer 로 받았다.

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
// * 느낀점

요본 문제는 생각보다 운이 좋았다.
생각한대로 풀려서 답이 나왔다.
그치만, 속도가 느려서 해결하느라 시간이 좀 더 걸린듯 하다.
매개변수 복사를 생각하자.
