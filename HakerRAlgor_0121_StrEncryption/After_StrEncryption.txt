// * 문제
StrEncryption

- 들어오는 문자열을 주어진 규칙으로 암호화해서 출력하는 문제.

- Input
w = string.
1 <= |w| <= 81

- Output
o = string. ( 암호화 된 문자열 )

- 암호화 방법.
strLen 의 루트를 구한 값으로,
행렬을 만들고,
Output 으로 만든 행렬의 열 기준으로 출력함.


/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
// * 해결 방법

1. strLen 의 해당 루트(strLen) 값을 구한다.

2. 해당 문자열을 행렬에 집어넣기. 
(** 열기준으로 집어넣는다. **)
for(j) - int colum = j % GridNum, CalcStr[colum].push_back;

3. Print - 열 기준으로 집어넣었으니.
해당 CalcStr[Num] 을 Append 하고, 
마지막을 제외하고, " " 을 붙여준다.
