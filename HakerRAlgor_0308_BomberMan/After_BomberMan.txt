// * 문제
- 문제이름 : BomberMan 폭탄 시뮬레이션

폭탄 - 'O'
빈 곳 - '.'

Input : Grid(vector<string>) , n(초)

상황 : 매 순간 초(n) 마다 Grid 에 놓여지는 폭탄 출력.

규칙
1. 초기에 임의의 Grid 판에 폭탄이 설치됨.
2. 1초 후, BomberMan 은 아무것도 하지않음.
3. 1초 후, 폭탄은 Cell 에 빈 곳이 존재하지 않도록 가득 채움.
4. 1초 후, 3초 전에 설치된 폭탄이 터짐.
5. 3,4 번 규칙을 반복.
*여기서 폭탄의 길이는 1 즉, 위 아래 양옆 만 터짐.

*목적 - n 초 후에 해당하는 Grid(vector<string>) 을 출력.

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
// * 해결 방법

규칙성을 찾는다.
1. n == 1일 때, 그냥 초기값.

2. n % 2 == 0 일 때, 즉, 짝수 값일 때는 폭탄이 가득 채워진다.

3. ((n - 3) % 4 == 0) 일 때,

4. ((n - 5) % 4 == 0) 일 때,

위 4가지 상황이 반복적으로 계속 나타남.

3, 4 번을 해결하기 위해 
Pair 를 선언. - int colum, int row
Grid 는 어차피 순차적으로 한번은 무조건 돌아야한다. 왜냐하면 'O' 을 찾아야되니까.
Grid 를 돌면서 'O' 를 찾고 'O' 로 가득 채워진 Grid 에서 찾은 Pair 를 돌면서
'.' 를 만듬.

이렇게 찾게 되면 3번 상황이 되고, 위의 과정을 한 번더 반복하게 되면,
4번 상황이 만들어짐.
